删除商品的方法
方法 1：使用删除脚本（推荐）
已创建删除脚本，可直接使用：
删除指定商品：
# 通过 ID 删除（例如删除 ID 为 1 的商品）
node scripts/delete-product.js 1
# 通过名称删除（例如删除"经典白衬衫"）
node scripts/delete-product.js "经典白衬衫"

查看所有商品：
node scripts/list-products.js

方法 2：修复 Prisma Studio
如果仍想使用 Prisma Studio，可尝试：
检查端口是否被占用
手动启动：
   npx prisma studio
然后访问 http://localhost:5555


方法 3：直接操作数据库文件
使用 SQLite 工具（如 DB Browser for SQLite）打开 prisma/dev.db 文件，直接删除记录。
